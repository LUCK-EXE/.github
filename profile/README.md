## 🎯 게임 개요
- **장르:** 경제 시뮬레이션 / 미니게임 컬렉션  
- **플랫폼:** PC (Unity 2D)  
- **핵심 컨셉:**  
  플레이어는 일정한 **자본**, **빚**, **이자율**을 가지고 시작해  
  하루에 3개의 미니게임(경마, 로또, 하이-로우, 채굴 노가다)을 플레이하며  
  **모든 부채와 이자를 상환**하는 것을 목표로 합니다.

<br>

## 💰 게임 목표
- 초기 자산으로 **모든 부채와 이자를 상환**하면 클리어  
- 자산이 0이 되어도 게임은 종료되지 않음 (**파산 없음**)  
- 핵심은 '생존'이 아니라 **'청산'**

<br>

## ⚙️ 시스템 개요
- 하루 = 3게임 진행 후 **이자 계산**  
- 이자율: **기본 10% (복리 적용)**  
- 자산이 `[부채 + 누적이자]` 이상이면 클리어  

<br>
